# Dashboard-Financeiro
Sistema de gerenciamento e visualização de estoque desenvolvido para uma mercearia familiar. O projeto utiliza JavaScript, Bootstrap e Chart.js para exibir métricas financeiras, gráficos interativos e relatórios baseados em planilhas Excel importadas pelo usuário.

## Funcionalidades

- Importação de arquivos Excel
- Dashboard responsivo
- Gráficos dinâmicos
- Controle de estoque
- Análise financeira
- Previsão de lucro
- Atualização automática dos dados
