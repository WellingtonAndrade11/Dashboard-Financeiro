# Dashboard de Estoque

Sistema web responsivo para controle de estoque e análise financeira de uma mercearia familiar.

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Chart.js
- SheetJS

## Funcionalidades

- Importação de arquivos Excel
- Dashboard responsivo
- Gráficos dinâmicos
- Controle de estoque
- Previsão de lucro
- Atualização automática de dados
